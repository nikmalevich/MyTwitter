Во всех запросах передаются 4 ключа для аутентификации, которые я указываю во вкладке Authorization в Postman 
ScreenName: получаем аккаунты пользователей, у которых в статусе есть слово snap. Для этого в параметрах к запросу, кроме ключей для авторизации, передаем для ключа screen_name значение snap
UserName: получаем аккаунты пользователей с именем nikita. Для этого в параметрах запроса, кроме ключей для авторизации, передаем для ключа q значение nikita
